# Cyber Soul
https://cybersoul.azulfolio.com/
## 概要
架空の公式アニメサイトです。
WordPressのオリジナルテーマで作成しております。
依頼を受けて制作するという想定で制作をしています。
レスポンシブ対応をしているので、スマホ・タブレットからもご確認いただけます。
## サイト構築
以下をChatGPTにて出力しました。
* 架空アニメ構想案
    * 題材やニュース文やキャラクター名など
* キャラクター画像などの一部画像

その他、サイトデザイン・コーディングについては独自で制作しています。
